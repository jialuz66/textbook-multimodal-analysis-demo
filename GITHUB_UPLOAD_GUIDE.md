## üöÄ Upload Steps

### Option A: Web Interface (Easiest)

#### Step 1: Go to Your Repository
Navigate to: `https://github.com/jialuz66/textbook-multimodal-analysis-demo`

#### Step 2: Delete Old File (if exists)
1. Click on `Enhanced_Prototype_4_Pages.html`
2. Click the trash icon (üóëÔ∏è) at top right
3. Scroll down, click "Commit changes"

#### Step 3: Upload New Files
1. Click **"Add file"** ‚Üí **"Upload files"**
2. Drag these files:
   - `index.html`
   - `README.md`
3. In commit message, type:
   ```
   Update prototype to V4 with 37 features and comprehensive documentation
   ```
4. Click **"Commit changes"**

#### Step 4: Enable GitHub Pages
1. Click **"Settings"** (top menu)
2. Click **"Pages"** (left sidebar)
3. Under **"Source"**:
   - Branch: **main**
   - Folder: **/ (root)**
4. Click **"Save"**
5. Wait 2-3 minutes for deployment

#### Step 5: Get Your Link
The page will show:
```
‚úÖ Your site is published at:
https://jialuz66.github.io/textbook-multimodal-analysis-demo/
```

### Option B: GitHub Desktop (More Control)

#### Step 1: Install GitHub Desktop
Download from: https://desktop.github.com

#### Step 2: Clone Repository
1. Open GitHub Desktop
2. **File** ‚Üí **Clone Repository**
3. Select `textbook-multimodal-analysis-demo`
4. Choose local folder
5. Click **"Clone"**

#### Step 3: Add Files
1. Open the cloned folder in File Explorer/Finder
2. Copy your files:
   - `index.html`
   - `README.md`
3. Paste into the folder

#### Step 4: Commit & Push
1. Return to GitHub Desktop
2. See your changes listed
3. In "Summary" box, type:
   ```
   Update to V4: 37 features + professional documentation
   ```
4. Click **"Commit to main"**
5. Click **"Push origin"**

#### Step 5: Enable Pages (if not already)
Follow Step 4 from Option A

## ‚ö†Ô∏è Common Issues & Fixes

### Issue 1: 404 Error
**Cause**: File not named `index.html`
**Fix**: Rename file to exactly `index.html` (lowercase)

### Issue 2: Page Not Updating
**Cause**: Browser cache
**Fix**: Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)

### Issue 3: Images Not Loading
**Cause**: Images are embedded as base64 (should work automatically)
**Fix**: Check console for errors (F12 ‚Üí Console tab)

### Issue 4: GitHub Pages Not Showing
**Cause**: Not enabled or wrong branch
**Fix**: Settings ‚Üí Pages ‚Üí Source: main ‚Üí Save
